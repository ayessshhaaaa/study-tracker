# 📚 Study Tracker

> A web-based productivity application designed to help students manage their studies, tasks, goals, and progress in one place.




\

## ✨ Features

* 🔐 User registration and login system
* ✅ Add, edit, update, and delete tasks
* 🎯 Goal tracking
* 📝 Notes section
* ⏰ Pomodoro timer
* 📅 Study calendar
* 📈 Progress tracking dashboard
* 📱 Responsive user interface

## 🛠️ Tech Stack

| Category | Technologies          |
| -------- | --------------------- |
| Frontend | HTML, CSS, JavaScript |
| Backend  | PHP                   |
| Database | MySQL                 |
| Tools    | XAMPP, phpMyAdmin     |

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/ayessshhaaaa/study-tracker.git
```

2. Start Apache and MySQL in XAMPP.

3. Create a MySQL database in phpMyAdmin.

4. Import your SQL file.

5. Update database credentials in `db_connect.php`.

6. Move the project folder to the `htdocs` directory.

7. Open:

```text
http://localhost/study-tracker
```
